# 东方Project人物识别系统
## 简介
本项目是北京邮电大学在第二届北京东方百校天则的摊位活动项目，采用YOLOv11模型对东方Project中的角色进行识别分类，实现了一个简单的角色识别系统。
本项目灵感借鉴[响子识图](https://www.remilia-scarlet.com/kyouko/)，作者[进度条](https://weibo.com/u/3184892462)。
